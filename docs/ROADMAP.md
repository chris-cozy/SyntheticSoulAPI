# Roadmap

This roadmap reflects active and planned directions for Synthetic Soul.

## Planned Features

- Handling multiple messages being sent before the agent finishes responding.
- Enhanced relationship graphs between multiple users
- Expanded multi-agent simulations
- Advanced long-term personality drift and adaptation
- More robust memory retrieval and relevance ranking
- Function-calling support for deliberate action execution
- Non-reaction-based messaging where agents can initiate outreach
- More advanced group chat context and participation logic
- Better handling for multiple user messages before a response is generated
- Agent-generated user personality profiles from repeated interactions
- Option for agents to schedule responses for later delivery

## How to Use This Roadmap

- Treat this as directional guidance, not a fixed release contract.
- Priorities may change based on stability, user feedback, or infrastructure constraints.
- API changes associated with roadmap items should follow the project versioning policy.
